This is an entry-level blog app which i practise PHP, MySQL and Redis with MVC folder structure.

Projede Neler Var?

* Kullanıcı kayıt ve giriş sistemi (session tabanlı)

* Profil sayfası (username, bio, avatar güncelleme)

* Post oluşturma, düzenleme ve silme (CRUD)

* Postlara görsel ekleme

* Beğeni sistemi (like / unlike – toggle)

* Kullanıcı takip etme / takibi bırakma

* Dashboard (tüm kullanıcıların postları)

* Feed (sadece takip edilen kullanıcıların postları)

* Post arama (başlık ve içerik üzerinden)

* Redis ile dashboard post cache

* MVC mimarisi (Controller / Repository ayrımı)

* PDO ile güvenli SQL sorguları

* Yetki kontrolü (sadece post sahibi edit/delete yapabilir)
